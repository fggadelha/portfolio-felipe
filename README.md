# 🌐 Portfólio - Felipe Gadelha

Este é o meu portfólio desenvolvido em **Next.js**, onde apresento meus projetos, habilidades e experiências como desenvolvedor web.

---

## 🚀 Tecnologias utilizadas
- [Next.js](https://nextjs.org/) – Framework React para aplicações modernas
- [React](https://react.dev/) – Biblioteca para construção de interfaces
- [TypeScript](https://www.typescriptlang.org/) – Tipagem estática para maior segurança
- [Tailwind CSS](https://tailwindcss.com/) – Estilização rápida e responsiva
- [Vercel](https://vercel.com/) – Deploy e hospedagem

---

## 📂 Estrutura do projeto
- `pages/` → Páginas principais do portfólio
- `components/` → Componentes reutilizáveis
- `public/` → Imagens e ícones
- `.gitignore` → Arquivos ignorados pelo Git
- `package.json` → Dependências e scripts

---

## ⚙️ Como rodar localmente
Clone o repositório e instale as dependências:

```bash
git clone https://github.com/fggadelha/portfolio-felipe.git
cd portfolio-felipe
npm install
npm run dev

O projeto estará disponível em http://localhost:3000
